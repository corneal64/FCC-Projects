# File Metadata Microservice Project

## Setup

Install the required node modules

```bash
npm install
```

Run the app

```bash
npm start
```

## Example Usage:

```
Cick "Browse...", Choose a file, Click "Upload File"
```

## Example Output:

```json
{
  "name": "my-logo.jpg",
  "type": "image/jpeg",
  "size": 428219
}
```
