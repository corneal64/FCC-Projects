# Timestamp Microservice

## Setup

Install the required node modules

```bash
npm install
```

Run the app

```bash
npm start
```

## Example Usage:

```
https://timestamp-app.something.com/December%2015,%202015
https://timestamp-app.something.com/1450137600
```

## Example Output:

```json
{ "unix": 1450137600, "natural": "December 15, 2015" }
```
