# URL Shortener Microservice

## Setup

Install the required node modules

```bash
npm install
```

Run the app

```bash
npm start
```

## Example creation

```url
localhost:3000/new/https://www.freecodecamp.com
```

## Example creation output

```json
{
  "original_url": "https://www.freecodecamp.com",
  "short_url": "localhost:3000/kZUrp"
}
```

## Usage

```url
localhost:3000/kZUrp
```

## Will redirect to

```url
https://www.freecodecamp.com
```

#### A solution to the [FreeCodeCamp's](https://www.freecodecamp.com) [URL Shortener Microservice Project](https://www.freecodecamp.com/challenges/url-shortener-microservice).
